Sinpro-bau

Sindicato dos Professores de Bauru | Full Stack & Cloud

Laravel Firebase Auth MySQL Firebase Storage Chatbot Vercel

🛠️ Tecnologias e Arquitetura

  • Backend & API: Desenvolvido com Laravel (PHP), utilizando uma arquitetura robusta para gerenciar o CRUD de conteúdos administrativos e notícias.
  • Cloud & Segurança: Integração com Firebase Authentication para gestão segura de associados e Firebase Storage para armazenamento escalável de documentos e mídias.
  • Banco de Dados: MySQL para persistência de dados administrativos e estruturação de informações do sindicato.

✨ Funcionalidades Principais

Autenticação

Login social e e-mail via Firebase, permitindo acesso exclusivo a áreas restritas para professores.

Gestão de Conteúdo

Painel administrativo completo para controle de notícias, convênios e comunicados oficiais.

Atendimento IA

Chatbot integrado para suporte rápido e triagem de dúvidas frequentes dos usuários.

Cloud Storage

Upload de documentos relevantes diretamente para a nuvem, facilitando o acesso móvel.
