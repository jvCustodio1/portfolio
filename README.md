## 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

* **HTML5 / CSS3** (Estrutura e Estilização)
* **Tailwind CSS** (Framework de estilização rápida e responsiva)
* **JavaScript (ES6+)** (Interatividade e lógica do menu)
* **AOS (Animate On Scroll)** (Animações de rolagem)
* **Google Fonts** (Tipografia Inter)

---

## 📂 Estrutura do Projeto
O projeto está organizado para facilitar a manutenção técnica:
* `index.html`: Página principal com as seções Home, Sobre, Projetos e Contato.
* `style.css`: Estilização personalizada e animações de fundo.
* `script.js`: Lógica do menu dropdown e inicialização de bibliotecas.
* `projetos/`: Pasta contendo as páginas de detalhamento para cada área (Power BI, E-commerce, WordPress).
